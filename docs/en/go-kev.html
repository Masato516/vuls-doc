go-kev

For detail of go-kev, see vulsio/go-kev

Usage: go-kev on different server

$ go-kev server -bind=192.168.10.1 -port=1328

Specify URL of the go-kev in config.toml

[kevuln]
url = "http://192.168.0.1:1328"

$ vuls report
